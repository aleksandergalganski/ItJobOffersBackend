# It Job Offers (Backend)

> Backend for ItJobOffers Frontend application written in Angular.

## Installation

Create file config.env in config folder and add following properties:

- NODE_ENV
- PORT
- MONGO_URI
- FILE_UPLOAD_PATH
- MAX_FILE_UPLOAD
- JWT_SECRET
- JWT_EXPIRE
- JWT_COOKIE_EXPIRE

'''
npm install
'''

## Run

'''
npm run dev
'''
